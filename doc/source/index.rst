Welcome to VUP's documentation!
===================================================================

VUP is a variable and unified particle framework created 2016 by me, Alexander Scheid-Rehder, for my bachelor thesis. It is meant to be flexible but also provides a lot of initial examples and simulations already. Expanding the framework and implementing more functionality is more than encuraged.
